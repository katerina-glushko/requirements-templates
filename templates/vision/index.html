  1. Твиты получаются только при открытии страницы. Пока не производится автоматическое обновление списка твитов.

  2. Твиты пока не подвергаются автоматической обработке.

3. To be

3.1. Схема

Схема описывает возможный вариант получения списков твитов для сведения Администратора.

3.2. Ограничения/Уточнения

  1. Пока не продумана обработка rate limits, предполагаем, что Администратор будет в тестовом режиме пользоваться доработкой.

3.3. Процесс "Получение твитов"

  1. Администратор открывает страницу со списком твитов в Web Application

  2. Web Application вызывает API Twitter
    GET /tweets
